# packge
c++
Qt(5.9.8)
基于TCP的局域网聊天小程序源码  包括服务端 和客户端 

服务端负责转发客户端发送过来的信息

包括登录界面 用户名密码使用了轻量级数据库SQLITE保存（登录与注册）

实现了群聊和私聊功能 （双击右侧列表显示的在线用户，即可自动弹出私聊窗口）
